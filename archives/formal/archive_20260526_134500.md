核心修改内容：
从上次正式归档至今的核心变更：

1. **新增 Megatron 训练配置**（swe_agent_blackbox_megatron.yaml）
   - Hydra defaults 中 override model_engine=megatron，自动加载 megatron_actor/critic/ref/engine 配置
   - actor 使用 Megatron 引擎：TP=4, sequence_parallel=true, use_distributed_optimizer=true, bfloat16, core_attn recompute
   - vanilla_mbridge=true，通过外部 mbridge 包实现 HF→Megatron 权重转换
   - 其余 rollout/data/algorithm/reward/agent_framework 配置与 FSDP 版一致

2. **新增 Megatron 启动脚本**（run_train_megatron.sh）
   - 基于 run_train.sh，新增 MEGATRON_TP/MEGATRON_PP/MEGATRON_CP 环境变量
   - 默认 --config-name=swe_agent_blackbox_megatron
   - 通过命令行 override 传入 actor.megatron.tensor_model_parallel_size 等参数

3. **Megatron 训练通路验证**
   - 解决 transformer_engine ABI 不兼容（源码编译 TE 2.15.0 匹配 PyTorch 2.10.0）
   - 解决 megatron-core 版本兼容性（megatron-core 0.17.0）
   - 解决 mbridge 不支持 qwen3_5（安装指定 commit 的 mbridge）
   - 解决 NCCL broadcast 失败（卸载冲突的 nvidia-nccl-cu13 包）
   - 解决 GPU OOM（开启 param_offload + optimizer_offload）
   - 最终验证通过：8 样本 agent 跑完 + reward 评估 + 权重更新成功
