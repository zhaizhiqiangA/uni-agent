核心修改内容（从上一次正式存档 archive_20260522_0315 至今）：

1. Docker ENTRYPOINT 修复：较新版本的 sweb 镜像设置了 `ENTRYPOINT ["/bin/bash"]`，导致 `CMD "sleep 2h"` 被当作 bash 参数而非直接执行。修复方式是在 `run_args` 前插入 `--entrypoint ""`，清除镜像默认入口点。

2. Gateway 轨迹日志：通过 `SWE_AGENT_LOG_TRAJECTORY=1` 环境变量启用，gateway 会在每条请求到达和响应返回时打印消息详情（role、tool_call_id、tool_calls 名称、content 前 300 字符、token 数量），便于调试 agent 与 LLM 的多轮交互。默认关闭，不影响正常性能。

3. 文档同步：DESIGN.md 和 GUIDE.md 均已更新反映上述变更。
