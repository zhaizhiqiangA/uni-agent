commit 描述：fix(swe_agent_blackbox): pass agent_config_path via tools_kwargs and add stage debug logging

核心修改内容：
推理测试暴露了 agent_config 传递链问题：

1. 根因：run_inference() 调用 swe_agent_runner 时，agent_config_path 不经过训练配置的 runner_kwargs，而是需要通过 tools_kwargs 传递。之前代码把已加载的 config dict 塞进 tools_kwargs["_agent_config"]，但 runner 只读 agent_config_path 参数，没读 tools_kwargs 里的值。

2. 修复：run_inference() 把 agent_config_path 路径字符串放入 tools_kwargs["agent_config_path"]；runner 优先读参数，fallback 到 tools_kwargs。mini_swe runner 不受影响（不读该字段）。

3. 调试日志：在 agent_runner 中添加 [sample N] phase=xxx 格式的阶段日志，便于定位 sandbox 启动、agent 执行、reward 评测各阶段。
